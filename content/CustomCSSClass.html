Расположение файла настройки шаблона

public/web/{NAMESPACE}/templates/{TEMPLATE_ID}/config/config.json

Структура файла

Для того чтобы добавить для блока BLOCK_KEYWORD, новые цсс классы

  1. Создаем файл public/web/{NAMESPACE}/templates/{TEMPLATE_ID}/config/config.json
  2. Добавляем общую секцию
  3. Добавляем в часть template->blocks, для необходимого блока

Пример файла config/config.json
