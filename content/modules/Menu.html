Модуль Меню

  • Стили и миксины модуля меню можно найти в локальном файле libs/application/less/modules/Menu.less

    LESS переменные

    В первую очередь меню стоит стилизировать через доступные переменные. Список переменных можно найти в локальном файле libs/application/less/variables.less.

    • @AppMod-Menu-Default-Indent - Вьюха default. Отступ между элементами меню.
    • @AppMod-Menu-Default-SelectSize - Вьюха default. Высота элемента select для мобильного устройства.
    • @AppMod-Menu-Default-SelectColor - Вьюха default. Цвет текста в элементе select для мобильного устройства.
    • @AppMod-Menu-Default-SeparatorWidth - Вьюха default. Ширина сепаратора, если указан класс has-separator.
    • @AppMod-Menu-Default-SeparatorHeight - Вьюха default. Высота сепаратора, если указан класс has-separator.
    • @AppMod-Menu-Default-SeparatorBackground - Вьюха default. Фон сепаратора, если указан класс has-separator.
    • @AppMod-Menu-Primary-Indent - Вьюха primary. Отступ между элементами меню.
    • @AppMod-Menu-Primary-SelectSize - Вьюха primary. Высота элемента select для мобильного устройства.
    • @AppMod-Menu-Primary-SelectColor - Вьюха primary. Цвет текста в элементе select для мобильного устройства.
    • @AppMod-Menu-Primary-SeparatorWidth - Вьюха primary. Ширина сепаратора, если указан класс has-separator.
    • @AppMod-Menu-Primary-SeparatorHeight - Вьюха primary. Высота сепаратора, если указан класс has-separator.
    • @AppMod-Menu-Primary-SeparatorBackground - Вьюха primary. Фон сепаратора, если указан класс has-separator.
    • @AppMod-Menu-Secondary-Indent - Вьюха secondary. Отступ между элементами меню.
    • @AppMod-Menu-Secondary-SelectSize - Вьюха secondary. Высота элемента select для мобильного устройства.
    • @AppMod-Menu-Secondary-SelectColor - Вьюха secondary. Цвет текста в элементе select для мобильного устройства.
    • @AppMod-Menu-Secondary-SeparatorWidth - Вьюха secondary. Ширина сепаратора, если указан класс has-separator.
    • @AppMod-Menu-Secondary-SeparatorHeight - Вьюха secondary. Высота сепаратора, если указан класс has-separator.
    • @AppMod-Menu-Secondary-SeparatorBackground - Вьюха secondary. Фон сепаратора, если указан класс has-separator.
    • @AppMod-Menu-Vertical-Indent - Вьюха vertical. Отступ между элементами меню.
    • @AppMod-Menu-Vertical-ChildIndent - Вьюха vertical. Отступ дочернего меню.

    Пользовательские вьюхи

    Иногда в зависимости от шаблона есть нужда переопределить текущую вьюху или добавить новую. Вьюхи находятся на сервере в директории module/system/Page/view/page/block/menu/templates. Заменять или добавлять новые вьюхи следует в директорию шаблона public/web/ID/templates/Template_ID/view/page/block/menu/templates, где ID - Microsite URL заполненный в форме создания микросайта, Template_ID - ID выбранного шаблона в форме создания микросайта.

    При написании стилей стоит учитывать, что у меню есть выравнивание и некоторые стили нужно прописывать в зависимости от него.
