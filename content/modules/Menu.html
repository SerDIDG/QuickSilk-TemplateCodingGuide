Модуль Меню

  • LESS переменные

    В первую очередь меню стоит стилизировать через доступные переменные. Их список можно найти в локальном файле libs/application/variables.less.

    • @AppModuleMenu-Default-Indent - Вьюха default. Отступ между элементами меню.
    • @AppModuleMenu-Default-SelectColor - Вьюха default. Цвет текста в элементе select для мобильного устройства.
    • @AppModuleMenu-Primary-Indent - Вьюха primary. Отступ между элементами меню.
    • @AppModuleMenu-Primary-SelectColor - Вьюха primary. Цвет текста в элементе select для мобильного устройства.
    • @AppModuleMenu-Secondary-Indent - Вьюха secondary. Отступ между элементами меню.
    • @AppModuleMenu-Secondary-SelectColor - Вьюха secondary. Цвет текста в элементе select для мобильного устройства.
    • @AppModuleMenu-Vertical-Indent - Вьюха vertical. Отступ между элементами меню.
    • @AppModuleMenu-Vertical-IndentX - Вьюха vertical. Отступ для вложенных элементов.

    Пользовательские вьюхи

    Иногда в зависимости от шаблона имеет смысл делать дополнительные вьюхи. При написании стилей стоит учитывать, что у меню есть выравнивание и некоторые стили нужно прописывать в зависимости от него.
